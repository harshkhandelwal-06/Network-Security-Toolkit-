# 🛡️ Network Security Toolkit

A comprehensive cybersecurity platform designed to help analyze, monitor, and secure network environments through a unified dashboard.

---

## 📖 Overview

Network Security Toolkit is a Python-based cybersecurity application that combines multiple security tools into a single platform. It enables users to monitor network activity, perform security assessments, analyze firewall configurations, and identify potential vulnerabilities.

---

## ✨ Features

### 🔍 Packet Sniffer

* Capture and analyze network packets
* Monitor incoming and outgoing traffic
* Real-time packet inspection

### 🚪 Port Scanner

* Scan open ports on target systems
* Identify active services
* Security assessment support

### 🔥 Firewall Rule Analyzer

* Analyze firewall configurations
* Detect conflicting rules
* Improve network security posture

### 🍯 Honeypot System

* Simulate vulnerable services
* Detect suspicious activity
* Log intrusion attempts

### 🌐 Attack Surface Analysis

* Domain and IP reconnaissance
* DNS and WHOIS lookups
* Basic OSINT capabilities

### 📍 IP Geolocation

* Identify geographic location of IP addresses
* Network information lookup
